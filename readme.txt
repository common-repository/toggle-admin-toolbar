== Description ==

Adds option to toggle admin toolbar visibility on WordPress frontend.

== Installation ==

1. Install plugin from WordPress plugin repository http://wordpress.org/plugins/
2. Activate Toggle Admin Toolbar through the 'Plugins' menu in WordPress.

= Manual Installation =

1. Upload the entire `toggle-admin-toolbar` directory to the `/wp-content/plugins/` directory.
2. Activate Toggle Admin Toolbar through the 'Plugins' menu in WordPress.
